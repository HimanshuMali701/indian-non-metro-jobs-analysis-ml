📊 Indian Non-Metro Job Market Analysis (EDA + ML)

📌 Project Overview

This project analyzes the Indian non-metro job market using exploratory data analysis and machine learning.
The objective is to uncover insights such as:

Salary patterns in Tier-2/Tier-3 cities

Role-based hiring trends

Skill influence on salary

Predictive modeling for salary estimation

🧠 Objective

✔ Clean & analyze job dataset
✔ Perform statistical & visual insights
✔ Train machine learning models to predict salary
✔ Evaluate model performance & extract feature importance
